Function Overloading in Java

One Name, Many Talents

The Story: The Smart Calculator

Imagine a calculator with just one button called `add`. This button is so smart it knows what to do based on what you give it. If you give it two numbers, it adds them. If you give it three numbers, it adds all three. If you give it decimal numbers, it adds those too! This is exactly how function overloading works in Java.

What is Function Overloading?

It's the ability to have **multiple functions with the same name** in the same class, as long as they have **different parameters**. Java is smart enough to call the correct one based on the arguments you provide.

Type 1: Different Number of Parameters

You can have methods with the same name but a different count of arguments.

// Adds two numbers
int add(int a, int b) { ... }

// Adds three numbers
int add(int a, int b, int c) { ... }

Type 2: Different Data Types of Parameters

You can also have methods with the same name and same number of arguments, but their data types are different.

// Adds two integers
int add(int a, int b) { ... }

// Adds two doubles (decimals)
double add(double a, double b) { ... }

Important Rule: Return Type Doesn't Count!

You **cannot** overload a function just by changing its return type. The method signature (name and parameters) must be different. Java won't know which method to call if only the return type is changed.

// This is perfectly fine
int add(int a, int b) { return a + b; }

// ERROR! Same name, same parameters.
double add(int a, int b) { return a + b; }

// This would cause a compile-time error.

The Smart Calculator Code

Here's how our calculator uses both types of overloading.

public class Calculator {

    // Method 1: Adds two integers
    public static int add(int a, int b) {
        return a + b;
    }

    // Method 2: Adds three integers (different number of params)
    public static int add(int a, int b, int c) {
        return a + b + c;
    }

    // Method 3: Adds two doubles (different type of params)
    public static double add(double a, double b) {
        return a + b;
    }

    public static void main(String[] args) {
        System.out.println("Adding 5 + 10 = " + add(5, 10));
        System.out.println("Adding 5 + 10 + 20 = " + add(5, 10, 20));
        System.out.println("Adding 5.5 + 10.5 = " + add(5.5, 10.5));
    }
}

Output:

Adding 5 + 10 = 15
Adding 5 + 10 + 20 = 35
Adding 5.5 + 10.5 = 16.0
