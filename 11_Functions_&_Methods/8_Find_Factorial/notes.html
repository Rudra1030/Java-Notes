Calculating a Factorial in Java

A simple guide to understanding and coding factorials.

What is a Factorial?

A factorial, shown by the symbol `!`, is the product of all positive integers up to a given number.

Formula: n! = n × (n-1) × (n-2) × ... × 1

Example: 4! = 4 × 3 × 2 × 1 = 24

Important Rules:

1! = 1 and 0! = 1 (by definition).

Java Program for Factorial

The Code (Iterative Method):

public class Factorial {
    public static void main(String[] args) {

        int number = 5; // The number to find the factorial of
        long factorial = 1; // Use 'long' for larger results

        // Loop from 1 to the number
        for(int i = 1; i <= number; i++) {
            factorial = factorial * i;
        }

        System.out.println("Factorial of " + number + " is: " + factorial);
    }
}

Code Breakdown:

1

We declare a `long` variable `factorial` and initialize it to `1`.

2

A `for` loop runs from `i = 1` up to our target `number` (5).

3

Inside the loop, we multiply the current `factorial` value by `i` and update it.

4

After the loop finishes, the final result is printed.

Program Output

Factorial of 5 is: 120
