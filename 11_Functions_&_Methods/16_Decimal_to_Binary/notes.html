Convert from Decimal to Binary

Translating human numbers into computer code with Java.

The Core Concept: Division and Remainders

To convert a decimal number to binary, you **repeatedly divide the number by 2** and record the remainder. The binary number is formed by taking all the remainders in **reverse order**.

Example: Decimal 5

5 / 2 = 2, Remainder = 1
2 / 2 = 1, Remainder = 0
1 / 2 = 0, Remainder = 1
Reading remainders backwards: 101

The Algorithm Step-by-Step

1

Start with a decimal number `n`.

2

Loop as long as `n` is greater than 0.

3

Inside the loop, find the remainder when `n` is divided by 2 (`rem = n % 2`).

4

Construct the binary number by adding the remainder at the correct place value.

5

Update `n` by dividing it by 2 (`n = n / 2`).

Java Program Implementation

public class DecimalToBinary {

    // Function to convert decimal to binary
    public static void decToBin(int n) {
        int myNum = n; // Store original number
        int power = 0;
        int binNum = 0;

        while (n > 0) {
            int remainder = n % 2;
            binNum += remainder * Math.pow(10, power);
            
            power++;
            n = n / 2;
        }

        System.out.println("Binary of " + myNum + " is " + binNum);
    }

    public static void main(String[] args) {
        decToBin(5);  // Example 1
        decToBin(8); // Example 2
    }
}

Program Output

Binary of 5 is 101
Binary of 8 is 1000
