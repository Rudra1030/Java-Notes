Binomial Coefficient (nCr) in Java

Finding the number of ways to choose 'r' items from a set of 'n'.

What is a Binomial Coefficient?

It tells you how many different ways you can choose a smaller set of items (`r`) from a larger set (`n`), where the order of selection doesn't matter.

Example: How many ways can you choose 2 players from a team of 5? This is written as 5C2.

The Mathematical Formula

nCr = n! / (r! * (n-r)!)

Where `n!` (n factorial) is `n × (n-1) × ... × 1`.

Java Program Implementation

The Code:

public class Binomial {

    // Helper function to calculate factorial
    public static int factorial(int n) {
        if (n == 0) return 1;
        int fact = 1;
        for (int i = 1; i <= n; i++) {
            fact = fact * i;
        }
        return fact;
    }

    // Function to calculate nCr
    public static int binCoeff(int n, int r) {
        int n_fact = factorial(n);
        int r_fact = factorial(r);
        int nmr_fact = factorial(n - r);

        int result = n_fact / (r_fact * nmr_fact);
        return result;
    }

    public static void main(String[] args) {
        int n = 5;
        int r = 2;
        System.out.println("C(" + n + ", " + r + ") = " + binCoeff(n, r));
    }
}

Code Logic:

To find the binomial coefficient, we simply follow the formula:

  1. Calculate the factorial of `n`.
  2. Calculate the factorial of `r`.
  3. Calculate the factorial of `n-r`.
  4. Divide the result of step 1 by the product of steps 2 and 3.

Example Calculation (n=5, r=2)

C(5, 2) = 10

So, there are 10 different ways to choose 2 players from a team of 5.
