Is the Number Prime?

A simple Java guide to identifying prime numbers.

What is a Prime Number?

A prime number is a whole number greater than 1 that has exactly two divisors: **1 and itself**.

Prime Examples: 2, 3, 5, 7, 11, 13

Not Prime Examples: 4 (divisible by 2), 6 (divisible by 2 and 3), 9 (divisible by 3)

The Logic to Check

1

Start with a number, `n`.

2

Loop from `i = 2` all the way to `n-1`.

3

In each loop, check if `n` is perfectly divisible by `i` (i.e., `n % i == 0`).

4

If you find even one number that divides `n` evenly, you can stop and say `n` is **not prime**.

5

If the loop finishes without finding any divisors, then `n` is **prime**!

Java Program Implementation

public class PrimeChecker {

    // A function to check if a number is prime
    public static boolean isPrime(int n) {
        // Corner case: Numbers less than 2 are not prime
        if (n <= 1) {
            return false;
        }

        // Check for divisors from 2 to n-1
        for (int i = 2; i < n; i++) {
            // If n is divisible by i, it's not prime
            if (n % i == 0) {
                return false;
            }
        }

        // If no divisors were found, it is prime
        return true;
    }

    public static void main(String[] args) {
        int num1 = 7;
        int num2 = 10;

        System.out.println(num1 + " is prime? " + isPrime(num1));
        System.out.println(num2 + " is prime? " + isPrime(num2));
    }
}
