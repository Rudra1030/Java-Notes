Convert from Binary to Decimal

Translating computer language to human language with Java.

The Core Concept: Place Value

Each digit in a binary number has a "place value" which is a power of 2. To convert, you multiply each binary digit by its place value and add them all up.

Example: Binary 101

( 1 × 2² ) + ( 0 × 2¹ ) + ( 1 × 2⁰ )
= ( 1 × 4 ) + ( 0 × 2 ) + ( 1 × 1 )
= 4 + 0 + 1 = 5

The Algorithm Step-by-Step

1

Start with two variables: `decimal = 0` and `power = 0`.

2

Loop as long as the binary number is greater than 0.

3

Inside the loop, get the last digit of the binary number (`lastDigit = binary % 10`).

4

Add `lastDigit * (2^power)` to your `decimal` total.

5

Increment the `power` by 1 and remove the last digit from the binary number (`binary = binary / 10`).

Java Program Implementation

public class BinaryToDecimal {

    // Function to convert binary to decimal
    public static void binToDec(int binNum) {
        int myNum = binNum; // Store original number
        int power = 0;
        int decNum = 0;

        while (binNum > 0) {
            int lastDigit = binNum % 10;
            decNum += lastDigit * Math.pow(2, power);
            
            power++;
            binNum = binNum / 10;
        }

        System.out.println("Decimal of " + myNum + " is " + decNum);
    }

    public static void main(String[] args) {
        binToDec(101);  // Example 1
        binToDec(1000); // Example 2
    }
}

Program Output

Decimal of 101 is 5
Decimal of 1000 is 8
